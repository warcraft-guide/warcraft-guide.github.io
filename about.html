About Us @ 27.12.2023 | mi.sh

Welcome to the Warcraft III Reforged Guide!
We think there aren't any good guides and tutorials on how to play this game, so it's very hard for newcomers to get into Warcraft, which is why many people won't experience this masterpiece.
That's why we've created this website. We want to create an easy-to-understand but comprehensive and in-depth guide for people new to Warcraft.
If you want to help us, join the Warcraft III Reforged Revolt Server or the Qyint Discord Server!
We have a dedicated channel for feedback, tutorial requests, etc.
Note that we aren't pro players, that's why we need your help to build this guide.
Huge thanks to everyone who contributed to this Site and I hope we could help you! ♡

Style Guide @ 27.12.2023 | mi.sh

This is a collection of tools for editing on this website.

We use bold text to highlight something.
We use italic and small text for comments.
We use red text for the most important things.
We use deleted text for outdated information. and subscript text for updates.
Example:
DO NOT CLICK THIS BUTTON OR YOU WILL DIE!
(for legal reasons: you won't.)
