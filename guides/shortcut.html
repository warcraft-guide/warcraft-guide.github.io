[ WARCRAFT III REFORGED GUIDE ]

Getting Started

? Create a Steam Shortcut for Warcraft III Reforged @ 28.12.2023 | mi.sh

Steam Deck users, and other users that like to have all their games in one place, might want to create a Steam Shortcut for Warcraft III Reforged. This will make it look like Warcraft is just another Steam game and it will allow you to start the game through Big Picture Mode, which is the Steam Decks default UI. You can also add custom controller layouts.

I. 1: Windows

Simply click Add a Game, then Add a Non-Steam Game... in Steam and add Warcraft III.
(You can use this method to add any application to Steam, it works on Linux too!)

I. 2: Linux

(This guide assumes you have installed Warcraft III Reforged according to our Installation Guide
Right-Click Battle.net in Lutris (or use the arrow button) and click Create steam shortcut.
You might have to restart Steam.

Screenshots:


II. Customization

Now you have a simple shortcut in Steam.
The only problem is that it does not have an icon, a banner or a cover.
You can solve this problem by visiting SteamGridDB, which is a website where users can upload artworks you can import to Steam. You can use this to customize other games too!
Just download whatever you want your Warcraft shortcut to look like!
To apply your custom artworks, open the shortcuts Properties so you can replace the icon and change the name.

Screenshots:


Next, right-click the (empty) banner to set a custom Background and Logo.
Once you have done that, you will get the Adjust Logo Position option too!

Screenshots:


And finally, right-click the shortcut in any category it's in to Set a custom artwork.
Remember that there are both horizontal and vertical Artworks.

Screenshots:


And that's it! You can now play W3R in Big Picture Mode, set a controller layout, etc.! Have fun! ♡
