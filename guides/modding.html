[ WARCRAFT III REFORGED GUIDE ]

Getting Started

? Mod Warcraft III Reforged @ 27.12.2023 | mi.sh

This guide will show you how to install the Quenching Mod for Warcraft III Reforged on Windows and Linux.
(The Linux Version of this guide assumes you have installed Warcraft III Reforged according to our Installation Guide

I. Downloads

Visit the Quenching Download Page and download both the Client and the Full Version.
Copy both Files to your Warcraft directory. By default, this is C:\Program Files (x86)\Warcraft III\ on Windows and $HOME/Games/battlenet/drive_c/Program\ Files\ (x86)/Warcraft\ III/ on Linux.
(The "Client" (the exe file) is the UI you need to configure the mod. The "Full Version" (the zip file) is the actual mod data. Technically you can download the mod through the client, but sometimes it just doesnt work, so this is the most reliable option.)

II. DLLs (Linux only, Windows users skip this step!)

On Linux, if you try to run the mod client right now, it will just throw a long error message at you. This is because, for some reason, there are two very important files missing. So if you're using Windows or you copied your game files from a Windows computer, you can skip this step. If you have a legal and safe way to get the CASCLib.NET.dll and ICSharpCode.SharpZipLib.dll files, you should do so. In case you can't, don't tell anyone I have you these.
Make sure these two files are in the Warcraft directory (see I.)

III. Installation

On Windows, just run the QMF2.2.exe. (the Client)

For Linux users: In Lutris, open the Configuration Menu and change the Executable to the QMF2.2.exe file. Save your configuration and hit Play.
(This menu is very important, remember how to change the Executable for later.)

Screenshots:



Two windows will open, one with the background video and one with the actual UI. You can minimize the video one.
If the video window appears on top of the UI and you can't move it out of the way, try toggling settings like Gamescope or the Virtual Desktop in the Configuration Menu.

Next, on both Windows and Linux, set your language to english and click Install Mod, then Manual Setup.
Find the mod data under C:\Program Files (x86)\Warcraft III\QMF2.2.zip and install it.
Once you see a window pop up telling you that the installation was successful, you can close this menu. (Now you can also delete the zip file.)

IV. Configuration

Click Settings and choose which GPU you're using. Everything else stays the same.
Click Change Theme to set your in-game background video. (Although I think this is broken on Linux)

IV. 4: Classic Graphics

If you want to use the Classic Graphics, click Settings, then Lock Classic. Make sure to also use the switch on the bottom-left of the in-game settings to enable Classic Graphics. You will have to do this every time you switch between Classic and Reforged.

V. More Info

Once you're done configuring Quenching Mod, you can close everything, on Linux, set your Executable back to Warcraft III Launcher.exe and start playing Warcraft!

If you are experiencing any other issues with Quenching, there is a Quenching Mod Guide Book that might help you.

I think the Campaign menu is useless, now that custom campaigns have been re-added to Warcraft III Reforged.
(Yes, I say RE-added. The original Warcraft III from 2002 already had custom campaigns. Reforged just got it a few updates ago.)

You are not logged in to Battle.net if you launch the game through the Quenching Client. This is why I'm telling you to set the Executable back to the Battle.net launcher.

And that's it! You modded Warcraft III Reforged! Have fun! ♡
