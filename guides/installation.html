Getting Started

? Install Warcraft III Reforged on Linux @ 27.12.2023 | mi.sh

Activision/Blizzard is not quite the best company when it comes to Linux and FOSS, so setting up Warcraft III Reforged hasnt been easy. This guide will show you how to install Warcraft III Reforged on Linux.
This guide was made with the Steam Deck in mind. I think Windows users already know what to do and can skip this entire page.

I. Get Lutris

You can download Lutris via Flatpak. On the Steam Deck, you can use Discover for this.

II. Launcher

Install Battle.net through the Lutris Installer. (this script)
(Do NOT use the Warcraft III Reforged Installer! Do NOT log into Battle.net yet, instead, close the Window once you're asked to log in!)

Screenshots:


III. Wine Configuration

Before doing anything else, open the Wine Configuration and change the Windows Version to Windows 7.

Screenshots:



IV. Warcraft

Now you're ready to launch Battle.net through Lutris!
Download Warcraft III Reforged through Battle.net!
The game should work as expected.

Or, if you already have the game installed somewhere on your computer, copy C:\Program Files (x86)\Warcraft III and C:\Users\[USER]\Warcraft III to the respective folders in your new Battle.net installation directory. By default this is $HOME/Games/battlenet/drive_c/... - then open the settings and scan for games.

And that's it! You can now play W3R on Linux! Have fun! ♡
