Setting up classic Warcraft III is not all that easy, since Blizzard took down the servers and all but one link to the installer. But it is, in a lot of ways, better than the Reforged version of the game. We have experienced lots of crashes, horrible performance - we literally can't even use the reforged graphics online, since they either drop our frames to 0 or crash the entire game. Reforged also breaks old custom maps and you don't own what you create in the World Editor anymore. On Linux, the World Editor doesn't even work properly. Its 30GB in size and costs 30€, rarely with 50% off. That's why many players want to just keep playing the classic versions. This guide will explain how you can install them.
This guide was tested on Linux with the Steam Deck in mind. This is an "incomplete" guide, as there is much more we could talk about. I. Download the Installers

You can download the Installers from Blizzard themselves. Links: Reign of Chaos & The Frozen Throne (Expansion).

II. Install the Game

Install Reign of Chaos by running the first installer.
If you use Linux, I would add it as a Non-Steam Game to my library and run it with Proton. (You could also add it as a locally installed game in Lutris)

The Installer will ask you for a key. If you have a Warcraft III Key, you can use that. (If you don't, you can simply use someone elses Key. You can always get yourself a Key later and change it.) You need both a RoC and a TFT Key, unless you don't want to install the Frozen Throne Expansion.

Once you have installed Reign of Chaos, install The Frozen Throne next. (If you use Linux, simply replace the executable in Steam or Lutris)

Yes! It's actually that easy! The entire Warcraft Community pretends like it's impossible!

III. Install Patches

Now we want to update to Patch 1.27b. Here is a ModDB backup of this old Patch, since Blizzard will force you to update all the way to Reforged and doesn't provide these old Patches. (Again, if you use Linux, simply replace the executable. If you used your own CD instead of these Installers, you might need some additional Patches.)

IV. Widescreen Fix

The game is really old, from a time before 16:9 displays were the standard. So if you want to use a "modern" 16:9 screen, you're either gonna have black bars or a really w i d e game. To fix this, we're gonna download the RenderEdge Widescreen Fix. Simply place it into the directory where your game is installed. (By default, this is C:\Program Files (x86)\Warcraft III)

V. Note:

Make sure to run Frozen Throne.exe to start the game and World Editor.exe to start the Editor. Custom Maps are obviously saved in the Maps folder.
